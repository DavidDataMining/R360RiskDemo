##关于模型...
现在有两个候选的包可供选择.
- XGBoost
- LightGBM.

现在用的是XGBoost, 不过无论从性能/训练速度/结果精度来看,最终都要更换为LightGBM.

有个好处是,两者都是基于GBDT的模型, 两者的参数可以互相转换. lgb最大的优势在于高内存效率的数据结构DataSet, 和高效的分裂树的方式.
